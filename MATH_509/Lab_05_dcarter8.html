Function for Problems 4 - 5¶

The next cell creates a Python function and a SymPy expression to represent the function $g(x, y)$ shown below. It also creates SymPy variables x and y and a list to contain those variables. Run this cell as-is.

$$\large g(x, y) = \frac{1000}{e^{(x + 2)^2 + 0.25 (y+1)^2}} + \frac{700}{e^{(x + 1)^2 + (y-3)^2}} - \frac{2000}{e^{0.25(x - 2)^2 + (y-1)^2}} + 10 (x + y)$$
In [8]:
def g(x,y):
    w1 = (x + 2)**2 + 0.25*(y + 1)**2
    w2 = (x + 1)**2 + (y - 3)**2
    w3 = 0.25*(x - 2)**2 + (y - 1)**2
    z = 1000/e**w1 + 700/e**w2 - 2000/e**w3 + 10*(x + y)
    return z

x, y = sym.symbols('x, y')
g_sym = g(x,y) * 2
g_vars = [x, y]

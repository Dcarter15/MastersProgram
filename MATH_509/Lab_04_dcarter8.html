Problem 6 - Calculating the Mean¶

Problem Background¶

Let $x_1, x_2, ..., x_N$ be a sample of numerical observations. The mean of such a sample can be calculated as follows:

$$\large \bar x = \frac{1}{N}\sum_{i=1}^N x_i$$

The mean of a sample has some interesting geometric properties. In particular, it is the unique value that minimize the total squared deviation with respect to the points in the sample. To state this explicitly, define the sum of squared deviations function, $SSD(x)$, as follows:

$$\large SSD(x) = \sum_{i=1}^N \left(x_i - x \right)^2$$

Then the mean $\bar x$ is the value of $x$ for which $SSD(x)$ is minimized. We will explore this aspect of the mean in this problem.

  3. Use display() to display the expression representing $SSD(x)$.

Problem 7 - Linear Regression Loss¶

Problem Background¶

In this problem, we will explore how calculus techniques can be used to find the optimal paramater values for a machine learning model. We will focus our attention on a simplified version of the linear regression model known as the no-intercept linear regression model. Assuming that our model has only one feature $x$, the no-intercept model can be written as follows:

$$\large \hat y = m\cdot x$$

The variable $m$ is parameter that defines the model. The goal during training is to find the value of $m$ that minimizes the sum of squared errors metric for some training set of $(x_i, y_i)$ feature/label pairs. The definition of $SSE(m)$ is provided below.

$$\large SSE(m) = \sum_{i=1}^N \left(y_i - \hat y_i \right)^2 = \sum_{i=1}^N \left(y_i - m\cdot x_i \right)^2 $$
